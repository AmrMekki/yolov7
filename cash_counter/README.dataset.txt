This projects combines the [Dollar Bill Detection project from Alex Hyams](https://universe.roboflow.com/alex-hyams-cosqx/dollar-bill-detection/) (`v13` of the project was exported in [COCO JSON format](https://roboflow.com/formats/coco-json) for import to this project) and the [Final Counter, or *Coin Counter*, project from Dawson Mcgee](https://universe.roboflow.com/coin-counter-v9-neqza/final-counter) (`v6` of the project was exported in [COCO JSON format](https://roboflow.com/formats/coco-json) for import to this project).

`v1` contains the original imported images, without augmentations. This is the version to download and import to your own project if you'd like to add your own augmentations.
